# Codecade Audio Assets

Place the following audio files in this directory:

## Required Files

1. **lofi-bg.mp3** - Calm lofi background music (2-5 min, will loop)
2. **click-soft.mp3** - Soft button click sound (50-150ms)
3. **hover-soft.mp3** - Subtle hover sound (30-100ms)
4. **scroll-soft.mp3** - Calm scroll sound (100-200ms)

## Free Audio Sources

**Music**: Pixabay, YouTube Audio Library, Free Music Archive
**SFX**: Freesound.org, Zapsplat.com, Mixkit.co

## Specs

- Format: MP3
- Bitrate: 128-192 kbps
- Sample Rate: 44.1 kHz

## Volume Settings (in audio-system.js)

- Background: 15%
- Click: 25%
- Hover: 15%
- Scroll: 20%
